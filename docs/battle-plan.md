## Risk Mitigation

### Risk: Cold email deliverability issues
**Mitigation:**
- Proper 2-week warmup before volume
- Multiple domains (don't put all eggs in one basket)
- Clean copy (no spam trigger words)
- Monitor bounce rates, pause if >5%

### Risk: Long sales cycle / slow responses
**Mitigation:**
- Persistent follow-up sequence (3 emails)
- Phone follow-up for hot leads
- Create urgency ("limited spots this month")

### Risk: Fulfillment bottleneck
**Mitigation:**
- Template-based builds (max 4-6 hours per site)
- Cap new clients at 5-6 per week initially
- Consider VA for admin tasks if overwhelmed

### Risk: Churn after launch
**Mitigation:**
- 12-month minimum commitment
- Deliver real value (sites that actually get results)
- Monthly check-ins with clients
- Quick response to update requests
