# 10K Project - Home Services WaaS

**Goal:** $10,000/month recurring revenue in 90-120 days
**Started:** March 22, 2026
**Budget:** $200

---

## The Model

Website-as-a-Service (WaaS) for home service businesses:
- Landscaping / Lawn Care
- Cleaning Services
- Pest Control
- Pressure Washing
- Pool Service

**Pricing:**
| Tier | Setup | Monthly | Target Clients for $10K |
|------|-------|---------|------------------------|
| Starter | $995 | $297/mo | 34 |
| Growth | $1,995 | $397/mo | 26 |
| Professional | $3,500 | $497/mo | 20 |

### Primary (Weeks 1-4)
