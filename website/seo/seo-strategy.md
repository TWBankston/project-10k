# SEO Strategy for TBDesigned.io

## Executive Summary
This document outlines the complete SEO strategy for TBDesigned.io, targeting home service contractors who need professional websites. Our goal is to rank for keywords that indicate high commercial intent from our ideal customers.

## On-Page SEO Checklist

### For Every Page
- [ ] Unique, keyword-rich title tag (under 60 chars)
- [ ] Compelling meta description (under 160 chars)
- [ ] H1 tag contains primary keyword
- [ ] H2-H3 tags use secondary keywords
- [ ] URL is short, descriptive, keyword-included
- [ ] Images have descriptive alt text
- [ ] Internal links to related pages (2-3 minimum)
- [ ] External links to authority sources (where appropriate)
- [ ] Content is 300+ words minimum (1000+ for blog posts)
- [ ] Mobile-friendly layout
- [ ] Fast loading (under 3 seconds)

### Homepage Specifics
- [ ] Primary service keywords in H1
- [ ] Location mentioned in first paragraph
- [ ] Clear value proposition above fold
- [ ] Trust signals (testimonials, logos, certifications)
- [ ] Clear CTA (call, form, or both)

### Service Pages
- [ ] One page per major service type
- [ ] Industry-specific keywords throughout
- [ ] Examples/portfolio for that industry
- [ ] FAQ section with schema markup
- [ ] Pricing information (or "starting at" pricing)
- [ ] Process explanation
- [ ] CTA specific to that service

### Blog Posts
- [ ] 1000-2000 words for ranking content
- [ ] Target one primary keyword + 2-3 secondary
- [ ] Include relevant images with alt text
- [ ] Link to service pages where relevant
- [ ] Include a CTA at end
- [ ] Update/refresh annually

```
│   └── /[post-slug]/
├── /about/
├── /contact/
└── /pricing/
```

### Technical Checklist
- [ ] SSL certificate installed (HTTPS)
- [ ] XML sitemap generated and submitted to Google Search Console
- [ ] robots.txt properly configured
- [ ] 404 page customized with navigation
- [ ] 301 redirects for any changed URLs
- [ ] Canonical tags on all pages
- [ ] Structured data/schema markup implemented
- [ ] Mobile-responsive design
- [ ] Core Web Vitals passing (LCP, FID, CLS)
- [ ] No broken links
- [ ] No duplicate content issues

### Speed Optimization
- [ ] Images compressed and WebP format where possible
- [ ] Lazy loading for images below fold
- [ ] CSS/JS minified
- [ ] Caching enabled
- [ ] CDN if needed
- [ ] Database optimized (WordPress)

## Backlink Strategy

### Phase 1: Foundation (Month 1-2)
1. **Google Business Profile** - Create/optimize listing
2. **Local directories** - Yelp, BBB, Asheville Chamber of Commerce
3. **Industry directories** - DesignRush, Clutch, UpCity, GoodFirms
4. **Social profiles** - LinkedIn, Facebook, Instagram (with website links)

### Phase 2: Content Marketing (Month 3-6)
1. **Guest posting** - Write for home service industry blogs
2. **HARO** - Respond to journalist queries for backlinks
3. **Infographics** - Create shareable visual content
4. **Partner links** - Exchange links with complementary businesses (not competitors)

### Phase 3: Authority Building (Month 6+)
1. **Case studies** - Publish and outreach to industry publications
2. **Original research** - "State of Contractor Websites" type content
3. **Tool/resource creation** - Free tools that attract links
4. **Local PR** - Asheville business features, events

### Link Targets
| Source Type | Target # Links | Priority |
|-------------|---------------|----------|
| Local directories | 20+ | HIGH |
| Industry directories | 10+ | HIGH |
| Guest posts | 5-10/year | MEDIUM |
| Resource mentions | Ongoing | MEDIUM |
| Local news/PR | 2-5/year | LOW |

### Link Building DO NOTs
- ❌ Buy links
- ❌ Link farms
- ❌ Excessive link exchanges
- ❌ Low-quality directory spam
- ❌ Irrelevant guest posts
