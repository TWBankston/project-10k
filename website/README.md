## Tech Stack

- **Platform:** WordPress (Hostinger)
- **Page Builder:** Elementor Pro
- **Theme:** Hello Elementor
- **Key Plugins:** ACF Pro, RankMath SEO, WP Rocket, Ultimate Member

## Key Differentiator: Client Portal

The client portal sets TBDesigned apart from competitors. It provides:

- Real-time project status and timeline
- Document/file upload system
- Deliverable review and approval workflow
- Support ticket system
- Post-launch credential access

See [dashboard/portal-spec.md](./dashboard/portal-spec.md) for full specification.
