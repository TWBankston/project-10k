## Image Optimization Guidelines

### Formats
- **JPG:** Photos, complex images
- **PNG:** Graphics with transparency, screenshots
- **SVG:** Icons, logos (vector)
- **WebP:** Modern format for web (with JPG fallback)

### Sizes
- Hero images: 1920x1080 max
- Content images: 1200px max width
- Thumbnails: 400px max
- Icons: SVG or 64px PNG

### Compression
- Use TinyPNG or Squoosh
- Target: Under 200KB for most images
- Hero images: Under 500KB
- Enable lazy loading for below-fold images

### Naming Convention
```
tbdesigned-[page]-[section]-[description].jpg
tbdesigned-hero-landscaping.jpg
tbdesigned-portfolio-sparkle-clean-desktop.png
tbdesigned-about-tyler-headshot.jpg
```
